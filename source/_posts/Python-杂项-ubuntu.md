## Python 注释

1. 使用 `#` 进行单行注释：

```python
# 单行注释
print("单行注释!")
```

2. 使用`三个连续的单引号`或`三个连续的双引号`进行多行注释：

```python
'''
多行注释！
多行注释！
三个连续的单引号！
'''
print("三个连续的单引号！")


"""
多行注释！
多行注释！
三个连续的双引号！
"""
print("三个连续的双引号！")
```

---

```python
#!/usr/bin/env python
# coding=utf-8

# 单行注释
print("单行注释!")

'''
多行注释！
多行注释！
三个连续的单引号！
'''
print("三个连续的单引号！")


"""
多行注释！
多行注释！
三个连续的双引号！
"""
print("三个连续的双引号！")
```

```bash
单行注释!
三个连续的单引号！
三个连续的双引号！
```
